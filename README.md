# DNP
Code for a 2,4 - Dinitrophenol visualizer </br>
Built off a slightly modified 3d "engine" I wrote a while back </br>
No dependencies, just clone repo and run </br>
Controls: a & d are Z axis rotation, g resets the camera, - & = zoom in and out </br>
Scale is if PPI is the real value. Default is 254 (macbook pro)
